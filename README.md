# 🛡️ Identify and Remove Suspicious Browser Extensions

## 📌 Objective
Learn how to identify potentially harmful or unnecessary browser extensions and remove them to improve browser security and performance.

---

## 🧰 Tools Used
- Google Chrome
- Internet Connection (for checking extension permissions & reviews)

---

## 🪜 Steps Performed

### 1️⃣ Open Extension Manager

#### ▶️ Google Chrome
- Open Chrome Browser  
- Type the following in the address bar:
  ```
  chrome://extensions/
  ```
- Press **Enter**
---

### 2️⃣ Review Installed Extensions

Carefully examine all installed extensions based on:

| Criteria        | Description                                   |
|----------------|-----------------------------------------------|
| Usage          | Is the extension actively used?               |
| Permissions    | Access to all websites or browsing data?      |
| Publisher      | Trusted developer or unknown source?          |
| Reviews       | Ratings and user feedback                     |
| Install Source | Installed manually or automatically?          |

---

### 3️⃣ Identify Suspicious Extensions

Suspicious extensions may:
- Request permission to read/change all website data  
- Have unknown publishers  
- Show low ratings or negative reviews  
- Redirect browser searches  
- Cause slow browser performance  
- Appear without user installation

---

### 4️⃣ Remove Suspicious Extensions

- Click on the **Remove** button next to the extension  
- Confirm removal  
- Restart the browser

---

---

## ⚠️ How Malicious Extensions Can Harm Users

Malicious extensions can:

- Track browsing history  
- Steal login credentials  
- Capture banking information  
- Display unwanted advertisements  
- Redirect to phishing websites  
- Access clipboard data  
- Slow down system performance  

Such extensions may act as spyware and transmit sensitive user data to third parties without consent.

---

## 🔄 Performance After Removal

After removing suspicious extensions:
- Browser startup time improved  
- Reduced unwanted pop-ups  
- Faster webpage loading  
- Improved overall browser performance

Suspicious or unused browser extensions may compromise user privacy and system performance. Identifying and removing such extensions enhances browser security and ensures a safer browsing experience.
